# WU Economics Tutor

Telegram-бот для подготовки к вступительному экзамену по экономике.

## Стек
- Python 3.12
- python-telegram-bot
- OpenAI Assistants API
- Vector Store (File Search)

## Запуск
1. Установить зависимости:
   pip install -r requirements.txt

2. Создать .env файл с переменными:
   TG_BOT_TOKEN=
   OPENAI_API_KEY=
   ASSISTANT_ID=

3. Запустить:
   python main.py
