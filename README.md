# 🎓 WU Economics Tutor

Telegram-бот с ИИ-ассистентом для подготовки к вступительным экзаменам по экономике.

---

## 🚀 Описание

**WU Economics Tutor** — это интеллектуальный Telegram-бот, который помогает изучать экономику, тренироваться и готовиться к экзаменам в интерактивном формате.

Бот использует **OpenAI API** для генерации ответов, объяснений и тестовых заданий.

👉 По сути — это персональный репетитор по экономике прямо в Telegram.

---

## 🎯 Возможности

* 📚 Объяснение экономических тем простым языком
* 🧠 Генерация вопросов и тестов
* 🎓 Мини-экзамены (симуляция экзамена)
* 🌍 Поддержка нескольких языков (RU / DE / EN)
* 🔁 Проверка ответов и объяснение ошибок
* 🤖 Общение с ИИ-ассистентом

---

## ⚙️ Как это работает

1. Пользователь пишет в Telegram-бот
2. Бот отправляет запрос в OpenAI API
3. ИИ формирует ответ
4. Ответ возвращается пользователю

Все происходит в реальном времени.

## 🔐 Переменные окружения

Создайте файл `.env`:

```
TG_BOT_TOKEN=your_telegram_token
OPENAI_API_KEY=your_openai_key
ASSISTANT_ID=your_assistant_id
```

## 🧑‍💻 Установка и запуск

### 1. Клонирование проекта

```
git clone https://github.com/ansavitskaya1973-prog/wu-tg-bot.git
cd wu-tg-bot
```

---

### 2. Создание виртуального окружения

```
python3 -m venv venv
source venv/bin/activate
```

---

### 3. Установка зависимостей

```
pip install -r requirements.txt
```

---

### 4. Запуск бота

```
python main.py
```

---

## ☁️ Деплой

Бот развернут на VPS (Beget):

* работает 24/7
* запускается как сервис (systemd)
* автоматически перезапускается

---

## 🛠️ Управление сервисом

```
systemctl start wu-tg-bot
systemctl stop wu-tg-bot
systemctl restart wu-tg-bot
systemctl status wu-tg-bot --no-pager
```
